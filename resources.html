Background colors are included in these layouts to show you where the div blocks are. You can remove or change the color.

If you want to use the .css for the div tags, go to the page you like, right click on page, select "Inspect", on the window that appears, select "Sources" menu, select the .css file, and copy the code from the window where it appears. Remember to remove the background color for each div when you have your page set.
