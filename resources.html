Remember to remove the background color for each div when you have your page set.

Additonal Tips

• How to access the source code for web pages:

Right click on the web page and select "Inspector", or on a Mac with a single button mouse, control + click and select "Inspector"

For Safari: Select the "Develop" link in the top Safari menu bar and select "Show Page Source".
To get the "Develop" optin in your menu, select the word "Safari in the upper left, select "Preferences...", select the "Advanced"/gear icon to the far right, click the box at the bottom of the screen, then close the Preferences window.
