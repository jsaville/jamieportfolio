Github Cheat Sheet

How to find your web site

https://username.github.io/repository/filename e.g. https://jsaville.github.io/jamieportfolio/truths.html

How to create a repository

  1. Go to Github.com
  2. Click on green ”New Repository” button
  3. Name your repository
  4. Select the “Public” option
  5. Click the “initialize repository with README”
  6. Select green “Create Repository” button

How to create a file in a repository

  1. Press “Create New File” within your repository
  2. Name your file
  3. Copy your code from Popcode to your new file in Github
  4. Add your css, javascript and jquery library links if needed (add css to head and js and jquery to body)
  5. Press ”Commit” to save the file

How to make your repository live

  1. Click on the “Settings/Gear” link in the upper right side of the screen
  2. Scroll down to “Source”
  3. Click on the “None” tab to reveal the “Master” options, select “Master Branch” and select “Save”
  4. How to take your web page offline
  5. Click on the “Settings/Gear” link in the upper right side of the screen
  6. Scroll down to “Source”
  7. Click on the “Master Branch” tab to reveal the “Master” options, select “None” and select “Save”

How to edit a file in your repo

  1. Click on the file name in the repo
  2. Click the edit tool (pencil) in the repo menu
  3. to save changes, “Commit”

How to edit your file name

  1. Click on the file name in the repo
  2. Find the file name and change it
  3. to save changes, “Commit”

How to copy code from someone else's repo

  1. You can access files on other repositories by going to their repo via their github repository URL e.g https://github.com/username/repositoryname https://github.com/jsaville/jamieportfolio
  2. Select the pencil/editing tool and copy the code
  3. Navigate back to your own repository (click the cat icon in the upper left corner of your page, then click on your repository)
  4. Open the file where you want the code, paste it and "Commit"
  5. You should see the hackone.html file in your repo

How to rename a file

  1. Click on the “Settings/Gear” link in the upper right side of the screen
  2. Scroll down to “Source”
  3. Click on the “Master Branch” tab to reveal the “Master” options, select “None” and select “Save”

How to delete a repo

  1. From the repo, click on the “Settings/Gear” link
  2. Scroll to the bottom of the page to the “Danger Zone”
  3. Select “Delete this repository”
